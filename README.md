To deploy this project run

```bash
  expo-cli start --tunnel
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/piug-07/Native-Luandry_app.git
```

Go to the project directory

```bash
  cd Laundry_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  expo-cli start --tunnel
```
